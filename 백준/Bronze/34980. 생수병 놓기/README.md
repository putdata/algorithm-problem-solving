## 문제 설명
민호는 학교에서 텀블러 대신 생수병을 가져오는 것으로 유명하다. 민호는 책상 위에 일렬로 붙어 있는 $N$개의 자리에만 생수병을 놓고, 절대 한 자리에 $2$개 이상의 생수병을 배치하지 않는 습관이 있다. 오늘도 민호는 $N$개의 자리 중 일부에 생수병을 배치해 두고 수업을 들으러 갔다.

수업이 모두 끝난 뒤 교실로 돌아온 민호는 자신의 생수병 배치가 이상하다는 느낌을 받았다. 민호는 아침에 어떻게 생수병을 배치했는지는 정확히 기억하고 있었지만, 혹시 생수병이 사라졌거나 바뀌었을지도 모른다는 불안감에 상황을 정확히 파악하지 못했다.

민호의 생수병 배치는 다음 네 가지 경우 중 하나로 분류된다.

- 생수병의 개수가 감소한 경우: `Oryang`
- 생수병의 개수가 같지만 배치가 달라진 경우: `Its fine`
- 생수병의 개수가 증가한 경우: `Manners maketh man`
- 생수병의 개수와 배치가 모두 동일한 경우: `Good`

혼란스러운 민호를 위해, 생수병에 어떤 일이 일어났는지를 판별하는 프로그램을 작성하자!

## 입력
첫번째 줄에 자리의 개수 $N$이 주어진다. $(1 \le N \le 150)$

두번째 줄에 아침에 민호가 배치한 생수병을 의미하는 길이 $N$의 문자열이 주어진다.

세번째 줄에 저녁에 민호가 확인한 생수병의 배치를 의미하는 길이 $N$의 문자열이 주어진다.

두번째 줄과 세번째 줄에 주어지는 문자열은 `w`와 `.`으로 구성되어 있으며, 문자열의 `w`는 생수병이 있는 자리, `.`는 비어 있는 자리를 의미한다.

## 출력
첫번째 줄에 생수병에 무슨 일이 일어났는지 알려주는 문자열을 출력한다.
